Conceptual data model ()
The conceptual model of DataCare is based on FHIR Resources v4.3.0. Our aim is to implement a detailed FHIR profile which can be used in all African countries. By default, base FHIR resources are used. Pregnancies are captured as Observations, following the FHIR International Patient Summary (IPS) standard (https://build.fhir.org/ig/HL7/fhir-ips/index.html). version: 0.2 last modified date: 03 October 2022
