Conceptual data model ()
The conceptual model of MomCare is based on FHIR Resources v4.3.0. Our aim is to implement a detailed FHIR profile which can be used in all African countries. An example of such a profile is the Dutch BirthCare 1.0 profile, see https://informatiestandaarden.nictiz.nl/wiki/Gebz:V1.0_FHIR_IG. version: 0.1 last modified date: 12 June 2022
